HTML

  • The head elements contains information about the webpage.
  • The body elements represents the visible content shown to the user.
  • The number order of headings matter, they must go in numerical order

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Different sections can be assigned different values such as font size, color and positioning.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • code .: opens up VS to be able to code in a GUI
  • cd "location" or cd: allows you to enter in the specified location or go back out
  • git add -A: to commit all files in the working branch
  • git commit -m "description of changes": adds a message to the commit
  • git pull origin main: pulls the base branch into current feature. Can display already up to date

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • In order to call apon a function, we need to list it as follows "functionName"()
